# 03 — 列表与计算属性

预期：输入框过滤下方标签列表。

```vue-interactive
<template>
  <div class="panel">
    <label class="label">
      筛选
      <input v-model="query" class="input" type="search" placeholder="输入关键字…" />
    </label>
    <ul class="list">
      <li v-for="item in filtered" :key="item" class="item">{{ item }}</li>
    </ul>
    <p v-if="filtered.length === 0" class="empty">无匹配项</p>
  </div>
</template>

<script setup lang="ts">
import { computed, ref } from 'vue'

const items = ['Vue', 'Obsidian', 'TypeScript', 'SFC', 'Sandbox']
const query = ref('')

const filtered = computed(() => {
  const q = query.value.trim().toLowerCase()
  if (!q) return items
  return items.filter((name) => name.toLowerCase().includes(q))
})
</script>

<style scoped>
.panel {

  flex-direction: column;

  max-width: 20rem;
}
.label {
  display: flex;

  gap: 0.35rem;
  font-size: 0.9rem;
}
.input {
  padding: 0.4rem 0.6rem;
  border-radius: 6px;
  border: 1px solid var(--background-modifier-border, #ccc);
}
.list {
  margin: 0;
  padding-left: 1.25rem;
}
.item {
  margin: 0.2rem 0;
}
.empty {
  margin: 0;
  opacity: 0.7;
  font-size: 0.9rem;
}
</style>
```
