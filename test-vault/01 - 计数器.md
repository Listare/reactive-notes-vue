# 01 — 计数器

预期：按钮可点击，`Count` 从 0 递增。

```vue-interactive
<template>
  <button class="counter-btn" type="button" @click="count++">
    Count: {{ count }}
  </button>
</template>

<script setup lang="ts">
import { ref } from 'vue'

const count = ref(0)
</script>

<style scoped>
.counter-btn {
  padding: 0.5rem 1rem;
  border-radius: 6px;
  border: 1px solid var(--background-modifier-border, #ccc);
  cursor: pointer;
  background: var(--interactive-accent, #7c3aed);
  color: var(--text-on-accent, #fff);
}
.counter-btn:hover {
  filter: brightness(1.05);
}
</style>
```
