# vue-interactive 手动测试索引

在 **阅读模式** 下打开下列笔记。开发时运行 `pnpm run dev`，保存源码后会自动重新编译并同步插件到本 vault，然后在 Obsidian 中重载插件或切换笔记即可查看效果。

- [[01 - 计数器]] — 按钮点击，数字递增
- [[02 - 作用域样式]] — 两块相邻组件，scoped 样式互不污染
- [[03 - 列表与计算属性]] — 过滤列表
- [[04 - 错误-非法导入]] — 应出现红色错误面板
- [[05 - 错误-缺少模板]] — 应出现红色错误面板
- [[06 - 导入示例]] — `@custom-script/` 导入 TS/CSS（需配置脚本路径 `scripts`）
- [[07 - 导入Markdown代码块]] — `?block=` 引用命名代码块
- [[08 - 自引用]] — 本笔记 `?block=`、命名 `vue-interactive`、循环依赖
- [[09 - 错误-递归过深]] — 应显示栈溢出错误面板，不崩溃
- [[10 - CDN导入]] — `https://` URL 从 esm.sh 导入（需联网）
- [[11 - Obsidian API]] — `@obsidian` 内置库（`app`、`Notice`）

- [[14 - MathJax]] — `@vue-interactive/math` 的 `Latex` 组件
